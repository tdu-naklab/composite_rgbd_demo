realsense_composite_demo
====
realsenseを使ったOC向け背景合成デモ

![gif](https://i.imgur.com/zEnUZ9P.gif)

## Required
- pipenv

pipenvを使わない場合は

- python >= 3.6.7
- pyrealsense2
- opencv-python

## Install
```bash
$ git clone git@github.com:tdu-naklab/realsense_composite_demo.git
$ cd realsense_composite_demo
$ pipenv install
```

## Run
```bash
$ pipenv shell
$ python ./src/demo.py
```
